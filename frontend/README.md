# Frontend

React + Vite + TypeScript. Команды — из корня репозитория или из этой папки:

```bash
npm install      # установить зависимости
npm run dev      # dev-сервер (http://localhost:5173), проксирует /api на uvicorn:8000
npm run lint     # oxlint
npm run build    # tsc -b && vite build
```

Подробности стека и решений — в `CLAUDE.md` и `DECISIONS.md` в корне репозитория.
